# Project: BullBear Mentors Trading Platform

## Overview
A MERN stack platform for selling trading courses with duration-based access and an admin dashboard for content and user management.

## Tech Stack
- **Frontend**: React (Vite)
- **Backend**: Express.js
- **Database**: MongoDB (Mongoose)
- **Auth**: JWT + Bcrypt
- **Styling**: Vanilla CSS (Premium, Dark/Light mode inspired)

## Core Features
1. **Auth System**: Login/Register, JWT protected routes. Priority: Login page (gatekeeper).
2. **Course Management**: Duration-based courses (1M, 6M, 12M) with discounts.
3. **Payment Verification**: Manual/Auto check via Transaction ID.
4. **Admin Dashboard**: CRUD for courses, success stories, certificates. User management (ban/block).
5. **Course Expiry**: Automated checks for course access duration.

## Theme & Design
- **Colors**: White (Dominant), Green (Profit/Success), Red (Loss/Caution), Black (Text).
- **Aesthetic**: Premium, clean, minimal but interactive.

## Directory Structure
- `/backend`: Express API
- `/Frontend/BullBear Mentors`: React Application
